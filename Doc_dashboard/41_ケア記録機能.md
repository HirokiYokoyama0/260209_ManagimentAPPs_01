## 1. 機能概要

### 1.1 解決したい課題

**患者側の課題:**
- 「今どの歯を治療しているのか分からない」
- 「次回は何をするのか覚えていない」
- 「これまでどんな治療をしたか忘れてしまう」

**スタッフ側の課題:**
- 「毎回カルテに詳細を書くのが大変」
- 「患者さんに説明する時間が取れない」
- 「過去の治療履歴を説明するのが手間」

### 1.2 実現したいこと

| 機能 | 説明 | 実現方法 |
|------|------|----------|
| **爆速タップ記録** | スタッフがキーボード入力なしで記録完了 | 歯の図をタップ + ステータスボタン1回 |
| **視覚的な治療履歴** | 患者が「マイ歯マップ」で治療状況を一目で確認 | 歯のイラスト + 色分け表示 |
| **タイムライン表示** | 「いつ・どこを・どう治療したか」を時系列で表示 | ケアの歩み（タイムライン） |
| **前回の記録を引き継ぎ** | 毎回ゼロから入力しない | 前回の状態をコピー + 差分だけ修正 |
| **キッズモード対応** | 乳歯のお子様にも対応 | 乳歯イラスト切り替え |

### 1.3 コンセプト

## 2. ユーザーストーリー

### 2.1 スタッフ（歯科衛生士・歯科医師）の視点

**シナリオ1: 虫歯治療の記録**

```
1. 患者一覧から「山田太郎」さんを選択
2. 「ケア記録」タブを開く
3. 前回の記録（右下4番が治療中）が表示されている
4. 今日は右下4番の治療が完了したので、その歯をタップ
5. [虫歯治療完了] ボタンを押す → 歯が緑色に変わる
6. メモ欄に「被せ物装着、経過良好」と音声入力
7. 「保存」ボタンを押す → 完了！（所要時間: 30秒）
```

**シナリオ2: 定期クリーニング**

```
1. 「佐藤花子」さんを選択
2. 「ケア記録」タブを開く
3. 今日は全体クリーニングなので、[全選択] ボタンを押す
4. [歯石除去] ボタンを押す → 全ての歯が水色に変わる
5. メモのテンプレートから「定期クリーニング実施」を選択
6. 「保存」→ 完了！（所要時間: 20秒）
```

### 2.2 患者の視点

**シナリオ1: 治療状況の確認**

```
1. LIFFアプリを開く
2. 「マイ歯マップ」タブをタップ
3. 自分の歯のイラストが表示される
4. 右下の歯が黄色（治療中）になっている
5. タップすると「虫歯治療中、次回で完了予定」と表示
6. 「あ、ここを治してるんだ」と理解できる
```

**シナリオ2: 過去の治療履歴を振り返る**

```
1. 「ケアの歩み」タブをタップ
2. タイムライン形式で過去の記録が表示される
   - 2026/03/06: 右下4番 虫歯治療完了
   - 2026/02/20: 歯石除去（全体）
   - 2026/01/15: 左上7番 被せ物装着
3. 「こんなに治療してもらったんだ」と実感
```

### 2.3 保護者（お子様の場合）の視点

**シナリオ: お子様の治療状況確認**

```
1. キッズモードで「たろうくん」のマイ歯マップを開く
2. 乳歯のイラスト（20本）が表示される
3. 右下の歯に「がんばったよスタンプ」がついている
4. 「次回は左上の歯をきれいにするよ！」とメッセージ
5. 保護者も治療の進捗が分かり安心
```

### 3.3 UI要素の詳細

#### 3.3.1 歯の選択

**表示方法:**
- 32本の歯をボタンまたはSVGで表示
- クリック/タップで選択状態に（背景色変更）
- 複数選択可能
- 前回の記録がある場合、その歯の色を変える（2026-03-07 更新）:
  - 🟢 緑 `#10b981`: 治療済み
  - 🟡 黄 `#eab308`: 経過観察
  - 🔴 赤 `#dc2626`: 治療予定
  - 🟠 橙 `#f97316`: 治療中
  - 🔵 青 `#3b82f6`: 被せ物
  - 🩵 水色 `#06b6d4`: 歯石除去済み
  - 🟣 紫 `#a855f7`: クリーニング済み
  - ⚫ 灰 `#9ca3af`: 未記録

**操作:**
- 個別クリック: その歯を選択
- Shift+クリック: 範囲選択
- 「全選択」ボタン: 全ての歯を選択
- 「クリア」ボタン: 選択解除

#### 3.3.2 ステータスボタン（2026-03-07 更新 - 実装済み）

**実装済みプリセット一覧（8種類）:**

| ボタンラベル | status値 | 色 | 説明 |
|------------|----------|-----|------|
| 虫歯治療済 | `cavity_completed` | 🟢 緑 `#10b981` | 虫歯の治療完了 |
| 経過観察 | `observation` | 🟡 黄 `#eab308` | 経過観察が必要 |
| 治療予定 | `cavity_planned` | 🔴 赤 `#dc2626` | 治療予定・要注意 |
| 治療中 | `in_treatment` | 🟠 橙 `#f97316` | 現在治療中 |
| 被せ物 | `crown` | 🔵 青 `#3b82f6` | クラウン・インレー装着済み |
| 歯石除去 | `scaling_completed` | 🩵 水色 `#06b6d4` | スケーリング完了 |
| クリーニング | `cleaning` | 🟣 紫 `#a855f7` | 予防クリーニング |
| クリア | `clear` | ⚫ 灰 `#9ca3af` | 記録をクリア |

**将来追加検討:**
- 抜歯 (`extraction`)
- 根管治療 (`root_canal`)
- 詰め物 (`filling`)
- ホワイトニング (`whitening`)

**動作:**
1. 歯を選択
2. ステータスボタンをクリック
3. 選択した歯に色がつく
4. 「保存」で確定

#### 3.3.3 メモ入力

**テンプレート:**
- 「定期クリーニング実施、問題なし」
- 「次回は○○の確認予定」
- 「経過良好、引き続き観察」
- 「被せ物装着、違和感なし」
- 「痛みあり、次回再診」

**音声入力:**
- ブラウザの音声認識API使用
- マイクボタンをクリックで起動
- 短い文章を音声で入力

**フリー入力:**
- テキストエリアで自由に入力
- 200文字程度まで

#### 3.3.4 前回の記録をコピー

**機能:**
- 前回の記録（tooth_data）を現在の状態として読み込む
- 変更箇所だけ修正すればOK

**使用例:**
```
前回: 右下4番が治療中（🟡）
今回: 右下4番をクリック → 「虫歯治療」ボタン → 緑に変わる
他の歯は前回のまま → 保存
```

## 5. データベース設計

### 5.1 patient_dental_records テーブル

**テーブル定義:**

```sql
create table patient_dental_records (
  id uuid primary key default gen_random_uuid(),
  patient_id text not null references profiles(id) on delete cascade,

  -- 歯の状態データ（JSONB）
  tooth_data jsonb not null default '{}'::jsonb,
  -- 例: {"44": {"status": "cavity_completed", "color": "green"}, "46": {"status": "cleaning", "color": "blue"}}

  -- 記録者・メモ
  staff_id uuid references staff(id) on delete set null,
  staff_memo text,

  -- 次回予定
  next_visit_memo text,

  -- 記録日時
  recorded_at timestamptz not null default now(),

  -- メタ情報
  created_at timestamptz not null default now()
);

-- インデックス
create index idx_dental_records_patient on patient_dental_records(patient_id);
create index idx_dental_records_date on patient_dental_records(recorded_at desc);

-- RLSポリシー
alter table patient_dental_records enable row level security;

-- anon keyで自分の記録のみ参照可能（アプリ層でpatient_idフィルタ）
create policy "anon_can_read_own_dental_records"
  on patient_dental_records for select
  using (true);

-- service_role（管理者）は全件操作可能
```

**DentalChart.tsx の実装例:**

```typescript
'use client';

import { useState } from 'react';

type ToothStatus = {
  status: string;
  status_label: string;
  color: string;
  updated_at: string;
};

type ToothData = {
  [toothNumber: string]: ToothStatus;
};

export default function DentalChart({
  initialData,
  onSave
}: {
  initialData: ToothData;
  onSave: (data: ToothData, memo: string) => void;
}) {
  const [toothData, setToothData] = useState<ToothData>(initialData);
  const [selectedTeeth, setSelectedTeeth] = useState<string[]>([]);
  const [memo, setMemo] = useState('');

  // 永久歯の配置（上下左右）
  const upperRight = ['18', '17', '16', '15', '14', '13', '12', '11'];
  const upperLeft = ['21', '22', '23', '24', '25', '26', '27', '28'];
  const lowerLeft = ['31', '32', '33', '34', '35', '36', '37', '38'];
  const lowerRight = ['41', '42', '43', '44', '45', '46', '47', '48'];

  const handleToothClick = (toothNumber: string) => {
    if (selectedTeeth.includes(toothNumber)) {
      setSelectedTeeth(selectedTeeth.filter(t => t !== toothNumber));
    } else {
      setSelectedTeeth([...selectedTeeth, toothNumber]);
    }
  };

  const handleStatusClick = (status: string, label: string, color: string) => {
    const updated = { ...toothData };
    selectedTeeth.forEach(toothNumber => {
      updated[toothNumber] = {
        status,
        status_label: label,
        color,
        updated_at: new Date().toISOString(),
      };
    });
    setToothData(updated);
    setSelectedTeeth([]); // 選択解除
  };

  const handleSave = () => {
    onSave(toothData, memo);
  };

  return (
    <div className="space-y-6">
      {/* 歯の配置図 */}
      <div className="border rounded-lg p-6">
        <div className="text-center mb-4">
          <p className="text-sm text-gray-600">上の歯</p>
          <div className="flex justify-center gap-1 mb-2">
            {upperRight.map(tooth => (
              <ToothButton
                key={tooth}
                toothNumber={tooth}
                isSelected={selectedTeeth.includes(tooth)}
                status={toothData[tooth]}
                onClick={() => handleToothClick(tooth)}
              />
            ))}
            <div className="w-4" /> {/* 中央の隙間 */}
            {upperLeft.map(tooth => (
              <ToothButton
                key={tooth}
                toothNumber={tooth}
                isSelected={selectedTeeth.includes(tooth)}
                status={toothData[tooth]}
                onClick={() => handleToothClick(tooth)}
              />
            ))}
          </div>
          <div className="border-t-2 border-gray-400 my-2" />
          <div className="flex justify-center gap-1 mt-2">
            {lowerRight.map(tooth => (
              <ToothButton
                key={tooth}
                toothNumber={tooth}
                isSelected={selectedTeeth.includes(tooth)}
                status={toothData[tooth]}
                onClick={() => handleToothClick(tooth)}
              />
            ))}
            <div className="w-4" /> {/* 中央の隙間 */}
            {lowerLeft.map(tooth => (
              <ToothButton
                key={tooth}
                toothNumber={tooth}
                isSelected={selectedTeeth.includes(tooth)}
                status={toothData[tooth]}
                onClick={() => handleToothClick(tooth)}
              />
            ))}
          </div>
          <p className="text-sm text-gray-600 mt-4">下の歯</p>
        </div>
      </div>

      {/* 選択中の歯 */}
      {selectedTeeth.length > 0 && (
        <div className="bg-blue-50 border border-blue-200 rounded-lg p-4">
          <p className="text-sm font-medium">
            選択中: {selectedTeeth.join(', ')}番
          </p>
        </div>
      )}

      {/* ステータスボタン */}
      <StatusButtonGroup onStatusClick={handleStatusClick} />

      {/* メモ入力 */}
      <MemoInput value={memo} onChange={setMemo} />

      {/* 保存ボタン */}
      <div className="flex justify-end gap-3">
        <button className="px-4 py-2 border rounded-lg">
          キャンセル
        </button>
        <button
          className="px-4 py-2 bg-green-600 text-white rounded-lg"
          onClick={handleSave}
        >
          保存
        </button>
      </div>
    </div>
  );
}
```
