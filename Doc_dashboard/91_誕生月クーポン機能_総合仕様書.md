**作成日**: 2026-03-28
**対象**: 管理ダッシュボード開発者・LIFFアプリ開発者・運用担当者

## 📋 機能概要

誕生月の患者に対して、LINE公式アカウントからFlex Message（カードタイプメッセージ）で特別クーポンを配信する機能です。

### 主な目的
- 誕生月というタイミングで患者エンゲージメントを向上
- 視覚的に訴求力の高いメッセージで来院促進
- プライバシーに配慮した誕生月のみの管理

---

## 🎂 クーポン内容

### 誕生月特別クーポン
- **割引内容**: 対象自費メニュー 10%OFF
- **有効期限**: 誕生月の月末まで
- **配信タイミング**: 毎月1日（その月が誕生月のユーザー全員に配信）
- **デザイン**: はぶらしかキャラクター付きカードメッセージ

### 1. データベース（Supabase）

#### マイグレーション
**ファイル**: `supabase/025_add_birth_month_column.sql`

```sql
-- birth_month カラム追加
ALTER TABLE profiles
  ADD COLUMN birth_month INTEGER DEFAULT NULL
  CHECK (birth_month >= 1 AND birth_month <= 12);

-- インデックス作成
CREATE INDEX IF NOT EXISTS idx_profiles_birth_month
  ON profiles(birth_month)
  WHERE birth_month IS NOT NULL;

-- 便利関数
CREATE OR REPLACE FUNCTION get_users_by_birth_month(target_month INTEGER)
RETURNS TABLE (...);

CREATE OR REPLACE FUNCTION get_current_month_birthdays()
RETURNS TABLE (...);
```

#### カラム仕様
- **カラム名**: `birth_month`
- **データ型**: `INTEGER`
- **制約**: `CHECK (birth_month >= 1 AND birth_month <= 12)`
- **NULL許容**: ✅ YES
- **値の意味**: 1=1月、2=2月、...、12=12月

### 誕生月別ユーザー数の確認

```sql
SELECT
  birth_month,
  COUNT(*) as user_count
FROM profiles
WHERE birth_month IS NOT NULL
  AND is_line_friend = true
GROUP BY birth_month
ORDER BY birth_month;
```

## 🔒 プライバシー配慮

### データ最小化の原則
- **保存するもの**: 誕生月（1-12の整数）
- **保存しないもの**: 誕生日、生年月日、年齢

### 理由
- 誕生月だけで誕生月クーポン配信の要件を満たせる
- 個人情報の取得を最小限に抑える
- プライバシーリスクを軽減

---

## ⚠️ 注意事項

### 既存ユーザーについて
- マイグレーション実行後、既存ユーザーの `birth_month` は `NULL`
- 誕生月クーポン配信の対象外となる
- LIFFアプリでプロフィール編集画面から後から登録可能

### 誕生月未登録のユーザー
- 誕生月フィルターで絞り込んだ場合、`birth_month = NULL` のユーザーは除外される
- プロフィール編集画面で「誕生月を登録すると、お誕生日クーポンが届きます！」などのメッセージで入力を促すことを推奨

### LINE友だち登録について
- 誕生月クーポンは**LINE友だち登録済みのユーザーのみ**に配信
- `is_line_friend = false` または `NULL` のユーザーは除外される

## 🛠️ トラブルシューティング

### Q: Vercel環境でFlex Message送信が失敗する
**A**: Flex Templateがコード内に埋め込まれているか確認してください。`lib/flex-templates/birthday-card.ts` が存在することを確認してください。

### Q: 誕生月フィルターで対象者が0人になる
**A**: 以下を確認してください：
1. 該当する月に誕生月のユーザーが登録されているか
2. `birth_month` が `NULL` ではないか
3. `is_line_friend = true` になっているか

### Q: 画像が表示されない
**A**: `public/images/はぶらしか_ハブラーシカ_キャラ.png` が存在し、正しくデプロイされているか確認してください。
