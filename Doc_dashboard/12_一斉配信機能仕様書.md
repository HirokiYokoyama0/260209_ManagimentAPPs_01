## 2. 新規配信タブ

### 2.1 セグメント条件選択

| 条件 | UI | 説明 |
|------|-----|------|
| **スタンプ数** | 数値入力（最小・最大） | 例: 最小 10、最大 20 |
| **最終来院日数** | 数値入力（最小・最大） | 例: 最小 90、最大 180（日） |
| **表示モード** | セレクトボックス | 全員 / 大人のみ / キッズのみ |
| **友だち登録** | チェックボックス | 「LINE友だち登録済みのみ」 |
| **誕生月** | 複数選択ボタン（12個） | 1月〜12月を複数選択可能（誕生月クーポン配信用） |
| **未入力項目** | チェックボックス（3項目、OR条件） | 氏名・診察券番号・誕生月が未入力の人を抽出 |

#### 未入力項目フィルター（詳細）

- **氏名が未入力の人**: `real_name` が空の患者を抽出（`display_name` はLINE自動取得のため必ず値あり）
- **診察券番号が未入力の人**: `ticket_number` が空の患者を抽出
- **誕生月が未入力の人**: `birth_month` が null の患者を抽出
- **OR条件**: 複数選択時、いずれか1つでも該当すれば抽出される

### 2.2 対象者プレビュー

- **ボタン**: 「対象者を確認」
- **表示内容**:
  - 対象者数: 「〇〇名」（除外後の人数を表示）
  - 除外人数: 個別除外した人数を表示
  - 推定メッセージ通数: 「〇〇通」（除外後の人数で計算）
  - **全員を表示・編集ボタン**: 対象者全員をモーダルで表示（最大500名）

#### 個別除外機能（全員表示モーダル）

- **機能**: 対象者全員をリスト表示し、個別に送信除外が可能
- **検索**: 氏名・診察券番号でリアルタイム検索
- **一括操作**: 「全員選択」「全員解除」ボタン
- **視覚表示**: 除外した人はグレーアウト + 「除外済み」バッジ
- **人数表示**: 除外人数と送信対象人数をリアルタイム表示
- **上限**: 対象者が500名を超える場合は警告メッセージを表示

### 2.3 メッセージタイプ選択

| タイプ | 説明 | 用途 |
|------|------|------|
| **テキストメッセージ** | シンプルな文字メッセージ | 通常の一斉配信、リマインド |
| **カードタイプメッセージ** | Flex Message形式のリッチカード | 誕生日クーポン、キャンペーン告知 |

### 2.4 メッセージ作成（テキストメッセージの場合）

| 要素 | 仕様 |
|------|------|
| **テキストエリア** | 1000文字まで |
| **変数挿入ボタン** | `{name}`, `{stamp_count}`, `{ticket_number}` を挿入 |
| **文字数カウント** | リアルタイム表示「〇〇 / 1000文字」 |
| **未設定ユーザー用配信文章ボタン** | 会員アプリ初期設定未完了向けのガイド文章を一括挿入（確認ダイアログ付き、緑系の色で区別） |

**変数の説明**:
- `{name}` → 患者の氏名（未設定の場合「お客様」）
- `{stamp_count}` → 現在のスタンプ数
- `{ticket_number}` → 診察券番号（未設定の場合「未設定」）
- `{valid_until}` → 有効期限（誕生月の月末日など）

### 2.5 メッセージ作成（カードタイプメッセージの場合）

| 要素 | 仕様 |
|------|------|
| **テンプレート選択** | 事前に登録されたテンプレート一覧から選択 |
| **プレビュー** | 選択したテンプレートの説明と使用可能な変数を表示 |

**利用可能なテンプレート**:
- **誕生日カード**: 誕生月特別クーポン（10%OFF）、はぶらしかキャラクター画像付き

### 2.6 送信実行

1. **送信ボタン**: 「〇〇名に送信」
2. **確認ダイアログ**:
   ```
   本当に送信しますか？
   対象者: 〇〇名
   メッセージ通数: 〇〇通

   [キャンセル] [送信する]
   ```
3. **送信処理**:
   - ローディング表示「送信中...」
   - 完了メッセージ「〇〇名に送信しました」
   - 自動的に配信履歴タブに切り替え

## 4. 技術仕様（簡略版）

### 4.1 データベース

#### broadcast_logs テーブル

```sql
CREATE TABLE broadcast_logs (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  sent_by TEXT NOT NULL,                    -- 送信者
  segment_conditions JSONB NOT NULL,        -- セグメント条件
  message_text TEXT NOT NULL,               -- メッセージ
  target_count INTEGER NOT NULL,            -- 対象者数
  success_count INTEGER DEFAULT 0,          -- 成功数
  failed_count INTEGER DEFAULT 0,           -- 失敗数
  sent_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_broadcast_logs_sent_at ON broadcast_logs(sent_at DESC);
ALTER TABLE broadcast_logs ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_public_read" ON broadcast_logs FOR SELECT USING (true);
CREATE POLICY "allow_public_insert" ON broadcast_logs FOR INSERT WITH CHECK (true);
```

### 4.2 API エンドポイント

#### `POST /api/broadcast/preview`

**Request**:
```typescript
{
  stampCount?: { min?: number; max?: number };
  lastVisitDays?: { min?: number; max?: number };
  viewMode?: 'adult' | 'kids' | null;
  isLineFriend?: boolean;
  birthMonths?: number[];  // 誕生月フィルター（例: [1, 2, 3]）
  missingFields?: {        // 未入力項目フィルター（OR条件）
    name?: boolean;        // 氏名（real_name）未入力
    ticketNumber?: boolean; // 診察券番号未入力
    birthMonth?: boolean;   // 誕生月未入力
  };
}
```

**Response**:
```typescript
{
  count: number;              // 対象者数
  profiles: Profile[];        // 全対象者（最大500名）
  estimatedCost: number;      // 推定通数
}
```

#### `POST /api/broadcast/send`

**Request**:
```typescript
{
  segment: BroadcastSegment;      // セグメント条件
  message: string;                // メッセージ（変数含む、textの場合のみ）
  sentBy: string;                 // 送信者名
  messageType?: 'text' | 'flex';  // メッセージタイプ（デフォルト: text）
  flexTemplateId?: string;        // Flex Templateの ID（flexの場合のみ）
  excludedIds?: string[];         // 個別除外する対象者のID一覧
}
```

**Response**:
```typescript
{
  success: boolean;
  broadcastLogId: string;
  targetCount: number;
  successCount: number;
  failedCount: number;
  error?: string;
}
```

#### `GET /api/broadcast/logs`

**Response**:
```typescript
{
  logs: Array<{
    id: string;
    sent_by: string;
    target_count: number;
    success_count: number;
    failed_count: number;
    sent_at: string;
    message_text: string;
    segment_conditions: BroadcastSegment;
  }>;
}
```

function replaceVariables(message: string, profile: Profile): string {
  return message
    .replace(/{name}/g, profile.display_name || 'お客様')
    .replace(/{stamp_count}/g, String(profile.stamp_count || 0))
    .replace(/{ticket_number}/g, profile.ticket_number || '未設定');
}

## 6. セグメント抽出ロジック

```typescript
function filterProfiles(profiles: Profile[], segment: BroadcastSegment): Profile[] {
  return profiles.filter(p => {
    // スタンプ数
    if (segment.stampCount?.min && p.stamp_count < segment.stampCount.min) return false;
    if (segment.stampCount?.max && p.stamp_count > segment.stampCount.max) return false;

    // 最終来院日数
    if (segment.lastVisitDays && p.last_visit_date) {
      const days = daysSince(p.last_visit_date);
      if (segment.lastVisitDays.min && days < segment.lastVisitDays.min) return false;
      if (segment.lastVisitDays.max && days > segment.lastVisitDays.max) return false;
    }

    // 表示モード
    if (segment.viewMode && p.view_mode !== segment.viewMode) return false;

    // 友だち登録
    if (segment.isLineFriend && p.is_line_friend !== true) return false;

    // 誕生月
    if (segment.birthMonths && segment.birthMonths.length > 0) {
      if (!p.birth_month) return false;  // 誕生月未設定は除外
      if (!segment.birthMonths.includes(p.birth_month)) return false;
    }

    // 未入力項目フィルター（OR条件）
    if (segment.missingFields) {
      const { name, ticketNumber, birthMonth } = segment.missingFields;

      if (name || ticketNumber || birthMonth) {
        let matchesAnyCondition = false;

        // 氏名（本名）が未入力
        if (name && !p.real_name) {
          matchesAnyCondition = true;
        }

        // 診察券番号が未入力
        if (ticketNumber && !p.ticket_number) {
          matchesAnyCondition = true;
        }

        // 誕生月が未入力
        if (birthMonth && p.birth_month == null) {
          matchesAnyCondition = true;
        }

        // いずれの条件にも該当しない場合は除外
        if (!matchesAnyCondition) return false;
      }
    }

    return true;
  });
}
```
